Partslingers Info:

Description:

Partslingers was a project I worked on for COMP 4081. The goal of this project was to create a convincing webapp that simulates a website to purchase PC parts. It included functionality for:

  • Searching

  • Browsing parts

  • Adding to cart

  • Checkout

  • Bundles

  • GPU comparison

  • Part checklist

  • Wattage calculator
